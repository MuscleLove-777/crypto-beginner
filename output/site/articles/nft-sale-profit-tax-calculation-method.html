NFT売却益の税金計算方法【初心者向け】確定申告ガイド

「NFT（非代替性トークン = 唯一無二のデジタルデータ）で得た利益には税金がかかるの？」「どうやって計算するの？」

この記事は、そんな疑問を持つNFT初心者の方のために、売却益にかかる税金の基礎知識から具体的な計算方法、確定申告の手順までを、暗号資産アナリストの視点から分かりやすく解説します。

この記事の結論

  • 結論1（最も重要なポイント）: NFTの売却益は、原則として「雑所得（ざつしょとく = 給与所得など他の所得とは異なる種類の利益）」に分類され、給与など他の所得と合算して課税されます。税率は累進課税（るいしんかぜい = 利益が多いほど税率も高くなる仕組み）が適用され、最大で所得税・住民税合わせて55%になる可能性があります。
  • 結論2: 売却益の計算には「移動平均法（いどうへいきんほう = 購入するたびに平均取得単価を計算する方法）」か「総平均法（そうへいきんほう = 年間の購入価格をまとめて平均取得単価を計算する方法）」のいずれかを選ぶ必要があります。一度選んだ計算方法は、原則として継続して使用しなければなりません。
  • 結論3: NFT取引の記録は、自分でしっかりと管理することが最も重要です。購入・売却価格、日時、数量、手数料（ガス代など）といった情報を正確に記録し、確定申告に備えましょう。国税庁のウェブサイトや、暗号資産損益計算ツールの活用が便利です。

基礎知識

NFTの売却益にかかる税金を理解するためには、まずいくつかの基礎知識を押さえておく必要があります。

NFTとは？その取引の仕組み

NFTは「Non-Fungible Token（ノンファンジブルトークン = 非代替性トークン）」の略で、「唯一無二の価値を持つデジタルデータ」のことです。デジタルアート、ゲーム内アイテム、音楽、デジタル会員証など、様々なものがNFTとして発行されています。これらは、ブロックチェーン（ぶろっくちぇーん = データを鎖のように繋げて記録していく、改ざんが非常に難しい技術）という技術基盤の上に成り立っており、所有者や取引履歴が明確に記録されます。

NFTの購入や売却は、主にイーサリアム（Ethereum）のような特定の暗号資産（あんごうしさん = インターネット上でやり取りされる、ブロックチェーン技術を用いたデジタルのお金）を使って行われます。例えば、OpenSea（オープンシー = 世界最大のNFTマーケットプレイス）などのプラットフォームで、ウォレット（うぉれっと = 暗号資産やNFTを保管するデジタルのお財布）を接続し、暗号資産を支払ってNFTを購入します。そして、そのNFTを別の誰かに売却することで利益を得る、というのが一般的な取引の流れです。

NFTの売却益は「雑所得」に分類される

日本では、NFTの売却によって得た利益は、原則として「雑所得」として課税されます。雑所得とは、給与所得、不動産所得、事業所得などの、他の9種類の所得のいずれにも該当しない所得の総称です。

  • 給与所得者（会社員など）の場合: 年間の雑所得が20万円を超えると確定申告が必要です。
  • 非給与所得者（個人事業主、フリーランスなど）の場合: 年間の所得全体で基礎控除額（きそこうじょがく = 全ての納税者に適用される所得控除。2023年時点では48万円）を超える利益が出た場合に確定申告が必要です。

雑所得は、給与所得など他の所得と合算され、所得税の「累進課税」が適用されます。つまり、利益が大きくなればなるほど、所得税率も段階的に高くなる仕組みです。所得税率は5%から45%まであり、これに加えて住民税10%が課税されるため、最大で55%の税金がかかる可能性があります。

損益通算について

残念ながら、NFT取引の雑所得は、株式投資（しゅしきとうし = 企業の株を売買して利益を得る投資）やFX（外国為替証拠金取引 = 外国通貨を売買して利益を得る取引）など、分離課税（ぶんりかぜい = 他の所得と区別して税金を計算する制度）が適用される所得との損益通算（そんえきつうさん = 利益と損失を相殺すること）はできません。ただし、同じ雑所得に分類される他の暗号資産取引の損失とは損益通算が可能です。

具体的な手順・やり方

NFTの売却益にかかる税金を計算し、確定申告を行うための具体的な手順を見ていきましょう。

ステップ1: 損益の計算方法を選ぶ

NFTの売却益を計算する方法には、「移動平均法」と「総平均法」の2種類があります。どちらの方法を選んでも構いませんが、一度選んだ方法は、原則として継続して使用する必要があります。

  • 移動平均法: NFTを購入するたびに、その時点での平均取得単価を計算する方法です。多くの取引を行う場合に、より正確な損益を把握できますが、計算が複雑になる傾向があります。
    • 計算例:
      1. 1月1日に1 ETHでNFT Aを購入 (1 ETH = 10万円)
      2. 2月1日に0.8 ETHでNFT Bを購入 (0.8 ETH = 9.6万円) → ここでETHの平均取得単価を再計算
      3. 3月1日にNFT Aを2 ETHで売却 (2 ETH = 25万円) 売却時のNFT Aの取得原価は、売却したNFT Aを構成するETHの平均取得単価で計算します。
  • 総平均法: 1年間の購入総額を、その年の購入総量で割って平均取得単価を算出する方法です。計算が比較的シンプルで、年間の取引が少ない場合に適しています。
    • 計算例:
      1. 1年間で合計5 ETHを様々な価格で購入し、総額50万円を費やしたとします。平均取得単価は1 ETHあたり10万円となります。
      2. この年、NFTを1 ETH分売却した場合、取得原価は10万円として計算します。

補足: どちらの計算方法を選ぶかは、税務署に「所得税の暗号資産の評価方法の届出書」を提出することで選択できます。提出がない場合は、自動的に「総平均法」が適用されます。

ステップ2: 必要な取引データを収集・整理する

損益計算には、すべての取引履歴が必要です。以下の情報を正確に記録・収集しましょう。

  • NFTの購入履歴: 購入日時、購入したNFTの名称、数量、支払った暗号資産の種類と数量、その時点での暗号資産の日本円での価格、手数料（ガス代 = ブロックチェーン上の取引手数料など）。
  • NFTの売却履歴: 売却日時、売却したNFTの名称、数量、受け取った暗号資産の種類と数量、その時点での暗号資産の日本円での価格、手数料。
  • 暗号資産の購入・売却履歴: NFTの購入に使用した暗号資産（ETHなど）を日本円で購入した際の履歴、または売却して日本円にした際の履歴も必要です。これは暗号資産自体の損益計算にも関わってきます。

これらのデータは、利用しているNFTマーケットプレイス（例: OpenSea）、暗号資産取引所（例: コインチェック、GMOコイン、DMM Bitcoinなど、日本の金融庁に登録されている取引所）、およびウォレットの取引履歴からダウンロードできることが多いです。ただし、複数のサービスを利用している場合は、それぞれの履歴を集約する必要があります。

便利なツール: 手動での計算は非常に手間がかかるため、「クリプタクト」や「Gtax」といった暗号資産専門の損益計算ツールを活用することをおすすめします。これらのツールは、複数の取引所の履歴を自動で取り込み、損益を計算してくれるため、大幅な時間短縮と計算ミスの防止につながります。

  • クリプタクト: 有料プランは月額1,980円から（年間契約の場合）。主要な国内・海外取引所やNFTマーケットプレイスに対応。
  • Gtax: 有料プランは月額1,980円から。こちらも幅広い取引所やDeFi（分散型金融 = 銀行を通さず直接お金のやり取りができる仕組み）に対応。

ステップ3: 確定申告書を作成する

収集・整理したデータをもとに、確定申告書を作成します。

  • 雑所得の計算: 損益計算ツールで算出した年間の雑所得（利益）の合計額を、確定申告書の「雑所得」欄に記入します。
  • 確定申告書B: 国税庁のウェブサイトで提供されている「確定申告書B」を使用します。e-Tax（いーたっくす = インターネットで確定申告ができるシステム）を利用すれば、自宅から手軽に申告が可能です。e-Taxの利用には、マイナンバーカードとICカードリーダー（またはスマートフォン）が必要です。

ステップ4: 申告・納税

確定申告の提出期限は、原則として毎年2月16日から3月15日です。期限に間に合うように準備を進め、忘れずに申告・納税を行いましょう。納税方法は、口座振替、クレジットカード払い、コンビニ払いなどがあります。{{internal_link:暗号資産の確定申告の期限と納税方法}}

暗号資産取引所の比較ポイント

NFTの購入に使う暗号資産（ETHなど）を入手するために、国内の暗号資産取引所を利用することが一般的です。以下の点を比較検討して選びましょう。

比較項目 国内取引所（例：コインチェック、GMOコイン） 海外取引所（例：Binance、Bybit）
金融庁登録 金融庁登録済み。日本の法律に準拠し、利用者保護が手厚い。 日本の金融庁に未登録の場合がある。自己責任での利用が求められる。
日本語対応 充実している。サポートも日本語で受けられる。 一部対応しているが、不完全な場合や専門用語の翻訳が不自然な場合も。
手数料率 日本円の入出金手数料や取引手数料が設定されている。 取引手数料は比較的安い場合が多いが、日本円入出金に手間がかかることも。

補足: NFTマーケットプレイス（OpenSeaなど）は、暗号資産取引所とは異なり、直接日本円での入金に対応していないことがほとんどです。そのため、まずは国内の金融庁登録済みの取引所で日本円を暗号資産に換え、その暗号資産をウォレット経由でマーケットプレイスに送金するという手順が必要になります。

リスクと注意点

NFT投資は大きなリターンが期待できる一方で、特有のリスクも存在します。税金対策と合わせて、しっかりとリスク管理を行いましょう。

  • 価格変動リスク: NFTの市場価格は、非常に流動的（りゅうどうてき = 変動しやすい）であり、予測が困難です。人気が急上昇したかと思えば、突然暴落することもあります。例えば、暗号資産市場全体でも、2022年には大手取引所FTXの破綻や、Terra/Luna（テラ/ルナ）の急落など、記憶に新しい暴落事例があり、NFT市場もその影響を強く受けやすい傾向にあります。「過去の実績は将来のリターンを保証しません」ので、常に慎重な判断が必要です。
  • ハッキング・詐欺リスク: NFTや暗号資産は、デジタル資産であるため、ハッキングや詐欺の標的になりやすいです。
    • フィッシング詐欺: 本物そっくりの偽サイトやメールで、ウォレットの秘密鍵（プライベートキー = 暗号資産やNFTの所有権を証明する最も重要な情報）やパスワードを盗み取ろうとする手口です。
    • ラグプル（Rug Pull）: プロジェクト運営者が、資金やNFTをだまし取って突然プロジェクトを放棄する詐欺です。
    • SCAM NFT（スキャムNFT）: 有名なNFTプロジェクトを模倣した偽物や、価値のないNFTを売りつける詐欺も横行しています。
  • 規制リスク: 暗号資産やNFTに関する法規制は、各国でまだ発展途上であり、変更される可能性があります。税制改正の動きにも常に注意を払いましょう。例えば、国際的な金融機関の監視団体であるFATF（金融活動作業部会）は、暗号資産に対する規制強化を各国に促しており、今後、より厳しいルールが導入される可能性もあります。
  • セキュリティ対策: ご自身の資産を守るためのセキュリティ対策は必須です。
    • 2段階認証（2FA = Two-Factor Authentication）: 暗号資産取引所やウォレットサービスでは、パスワードだけでなく、スマートフォンアプリなどで生成されるコードも入力する2段階認証を必ず設定しましょう。
    • 秘密鍵（プライベートキー）の管理: ウォレットの秘密鍵（またはシードフレーズ = 秘密鍵を復元するための単語の羅列）は、決して他人に教えず、オフラインで厳重に保管してください。ハードウェアウォレット（はーどぅぇあうぉれっと = USBメモリのような物理的なデバイスで暗号資産やNFTを保管する方法）の利用も強く推奨されます。
    • パスワードの強化: 長くて複雑なパスワードを設定し、使い回しは絶対に避けましょう。

投資は自己責任です。余裕資金の範囲内で行いましょう。

まとめ：初心者が今すぐやるべきこと

NFTの売却益にかかる税金について理解を深めていただいたところで、初心者の皆さんが今すぐやるべきことを3つのアクションにまとめました。

  1. 損益計算方法を決めて、記録を始める: 移動平均法か総平均法か、どちらで計算するかを決め、今日からでもNFTの購入・売却履歴、手数料（ガス代を含む）、その時点での日本円価格などのデータを詳細に記録し始めましょう。スプレッドシートでも構いませんが、専門の損益計算ツールを導入するのが最も効率的です。
  2. 取引所のセキュリティ設定を徹底する: NFTの購入に必要な暗号資産を扱う国内の金融庁登録済み取引所の2段階認証は必ず有効にし、ウォレットの秘密鍵（シードフレーズ）は安全な場所（できればオフラインのハードウェアウォレットなど）に保管しましょう。詐欺やハッキングから資産を守る第一歩です。{{internal_link:安全なウォレットの選び方と秘密鍵の管理}}
  3. 税制改正や市場の動向にアンテナを張る: 暗号資産・NFTに関する税制はまだ発展途上にあり、今後も変更される可能性があります。国税庁のウェブサイトや信頼できるメディアからの情報収集を習慣にし、最新の情報を常に把握するよう努めましょう。

まずは、この記事を参考に、ご自身のNFT取引の記録を見直すところから始めてみてください。適切な知識と準備で、安心してNFTの世界を楽しみましょう！
